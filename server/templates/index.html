Welcome to the Tiny Feet GHG Database Reader

This app can read all entries for each table in the database.

This app uses flask and sqlalchemy to connect to our MySQL database using read-only permissions, currently hosted on bluehost

This site uses jinja for template inheritence and for passing python dictionaries of the table rows and columns
queried from sqlalchemy Object Relational Mapping classes into the html template to be iterated over in a for loop and displayed in html tables

The tables are all queried at the same time, but there is a javascript function that unhides the tables when selected from the drop-down menu
